# Daten

Die ursprünglichen Excel- beziehungsweise CSV-Dateien sind nicht Teil dieses Repositories.

Gründe:

- Fokus des Portfolio-Projekts auf Datenmodell, SQL und Dashboard
- keine ungeklärte Weitergabe fremder oder lizenzierter Rohdaten
- Vermeidung unnötig großer Dateien im Repository

Für eine Reproduktion werden vier Tabellen mit den in [`docs/data-model.md`](../docs/data-model.md) beschriebenen Spalten benötigt:

- `daily_sales`
- `retailers`
- `products`
- `methods`
